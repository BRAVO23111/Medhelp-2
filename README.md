# MedHelp

MedHelp is a web application designed to simplify the process of booking appointments with local doctors. Users can easily book appointments online without needing to visit clinics. The platform provides features for users, doctors, and administrators to manage appointments and schedules efficiently.

## Features

- **User Authentication**: Users can register and log in to the platform.
- **Appointment Booking**: Users can browse and book available time slots with doctors.
- **Admin Control**: Administrators can manage doctors and their schedules.
- **Doctor Dashboard**: Doctors can view their upcoming appointments.
- **Future Enhancements**:
  - **Disease Prediction Software**: Auto-generates prescriptions based on symptoms.
  - **Cloud Storage**: Centralized storage for prescriptions and medical records.

## Technologies Used

- **Frontend**: 
  - React (Vite)
  - Tailwind CSS
  - Recoil (State Management)
- **Backend**: 
  - Node.js
  - Express.js
- **Database**: 
  - MongoDB

## Installation

To set up the project locally, follow these steps:

1. **Fork the Repository**: Click on the "Fork" button at the top right of this repository.

2. **Clone the Repository**:
   ```bash
   git clone <your-forked-repo-url>
   ```

3. **Navigate to the Project Directory**:
   ```bash
   cd MedHelp
   ```

4. **Install Dependencies**:
   - For the frontend:
     ```bash
     cd client
     npm install
     ```
   - For the backend:
     ```bash
     cd server
     npm install
     ```

5. **Start the Application**:
   - For the frontend:
     ```bash
     npm run dev
     ```
   - For the backend:
     ```bash
     npm start
     ```

## Usage

- **Users**: Register or log in to book appointments with available doctors.
- **Doctors**: Log in to view upcoming appointments and manage your schedule.
- **Admins**: Manage doctors' availability and schedules.
